# 🎬 SerieSN Explorer

Application de découverte des séries sénégalaises.

## Description

SerieSN Explorer permet de parcourir un catalogue de séries sénégalaises, consulter leurs détails, gérer ses favoris et filtrer par genre ou par chaîne. L'application fonctionne entièrement hors-ligne.

## Fonctionnalités

- Catalogue de 13 séries sénégalaises
- Recherche en temps réel par titre
- Filtres par genre et par chaîne
- Page détail avec synopsis et acteurs
- Système de favoris persistant
- Mode sombre / clair
- Note interactive par étoiles

## Installation

```bash
npm install
npm run dev
